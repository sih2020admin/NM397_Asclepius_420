# NM397_Asclepius_420

Voice based navigation for Bhuvan Portal

## Installation
Clone the repo and run 'npm i' 
'npm start'

## Working Commands
### English
Go up
Go down
Go left
Go right
Zoom in
Zoom out
Go to Delhi, Nagpur, Bhopal, Indore, Goa

### Hindi
Upar jao
Neeche jao
Baye jao
Daye jao
Bada karo
Chota karo
Delhi chalo
Nagpur chalo
Bhopal chalo
Indore chalo
Goa chalo
